# Vibhor Saxena — Portfolio

A modern, fully responsive personal portfolio showcasing my projects, skills, experience, and competitive programming journey. Built with performance, clean UI, and smooth animations as core priorities.

Live Website: https://portfolio-vibhor-3vfe.vercel.app  
GitHub: https://github.com/VibhorSaxena-23

---

## Features

- Built with Next.js (App Router)
- Fully responsive and mobile-optimized
- Custom animated cursor (desktop only)
- Smooth animations using Framer Motion
- Clean section-based layout
- Dark-themed modern UI
- SEO-friendly metadata and viewport handling
- No mobile zoom or horizontal overflow

npm run dev
